# AlphaDebater
🤖 AlphaDebater is a light-weight, adversarial multi-agent stock prediction system powered by LangGraph and DeepSeek-R1 / GPT-4o.

Instead of generated-by-LLM summaries, it simulates a Wall Street investment committee: a Bull Agent and a Bear Agent go through 3 rounds of brutal debate over the same financial data, forcing the CIO Agent to mathematically evaluate the confidence score and calculate the final target price.
